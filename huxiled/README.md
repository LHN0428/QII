  * huxiled.v文件是四位LED呼吸灯的代码
  
  ******************************************************************************************************
  * 四个呼吸灯的变化为：
  * 第1个灯隔1秒，亮1秒后变暗；然后第2个灯隔1秒，亮2秒后变暗；
  * 然后第3个灯隔1秒，亮3秒后变暗；最后第4个灯隔1秒，亮4秒后变暗。之后循环往复
  ******************************************************************************************************
  
  * 端口定义：需要四个端口(clk ,rst ,led[3:0])
